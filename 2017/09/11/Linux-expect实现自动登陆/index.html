expect详解

expect是一个用来处理交互的命令。借助expect，我们可以将交互过程写在一个脚本上，使之自动化完成。expect中最关键的四个命令是send expect spawn interact。

  • send：用于向进程发送字符串
  • expect：从进程接收字符串
  • spawn：启动新的进程
  • interact：允许用户交互

send命令接收一个字符串参数，并将该参数发送到进程：

expect

expect用来等待一个进程的反馈，它可以接收一个字符串参数，也可以接收正则表达式参数，与send命令结合，可以实现一个简单的交互：

spawn

spawm命令用来启动新的进程。spawn之后的send与expect命令都是与spawn所打开的进程进行交互。

interact

interact命令用来保持交互状态，把控制权交给控制台，若没有这句操作命令，程序执行完成之后就会退出，而不会留在远程终端上。
